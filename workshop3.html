Why Python

You've got a shell and a repo. Now the language that turns them into science.

Python is…

High-levelyou don't manage memory — you describe the idea
Interpretedruns without compiling; you can poke at it line by line
Readableclear, minimal syntax — good Python often reads like pseudocode
Cross-platformthe same code runs on your laptop and on Fiji

Large ecosystemNumPy, pandas, SciPy, BioPython, scikit-learn — someone has already written the hard part
Fast prototypingtry an idea before lunch, iterate after
Communityvast user base, excellent docs, an answered Stack Overflow question for nearly everything
Glue languagebridges C++, Java, R, and bash — it holds pipelines together

Speak Python

Third language, same lesson. Bash gave you imperative verbs; git gave you two-word verbs. Python gives you full sentences — with a subject, a verb, and a rule that whitespace is grammar. Tap any word to hear it in English.

verb / functionkeywordvalue name (variable)grammar particles

counts[nuc] += 1

def translate(dna):
return peptide

The little words — Python's particles

= vs ==“let this name mean…” vs “are these equal?” — the most common beginner bug in any language
: + indentthe colon opens a block; the indentation is the block. Python has no braces — whitespace is syntax
#“ignore the rest of this line” — a note to humans
[a:b]a slice: from a, up to (not including) b. seq[:5] first five · seq[-1] last one · seq[::-1] reversed
f'…{x}…'an f-string: drop the value of x right into the text
.method()“ask this object to do something to itself” — line.strip(), dna.replace('T','U')
The [start:end:step] slice is the sentence you'll use most today. sequence[::-1] reads “everything, backwards” — half of a reverse complement, in nine characters.

      STEP 5Transcribe & translate
      
      
      
    

    
    

    

    
hint
Transcription is one call: dna.replace('T','U'). Then walk the RNA
      three letters at a time — range(0, len(rna)-2, 3) — looking each codon up in codon_map
      and breaking on 'Stop'. Test it on 'ATGTCGTAA': you should get 'MS'.
      sequence.find('ATG') finds the first start codon.

    
the classic bioinformatics bug 🐛
The genome's first ATG (index 30)
      translates to a 16-residue nothing. A start codon is not a gene. Real gene coordinates come from a genome's
      annotation (GenBank lists lambda's genes on the
      J02459.1 record) —
      lambda's first gene, Nu1, is annotated 191..736.
      Watch out: GenBank counts from 1 and includes both endpoints; Python counts from 0 and excludes the end.
      So 191..736 is sequence[190:736] in Python. Off-by-one here silently
      frameshifts your protein into gibberish — check that your slice actually starts with ATG before
      you believe the output. Translate from index 190 and you get Nu1, the 181-residue terminase small subunit:
      a real protein, found with a for-loop.

      C1Reverse translation
      
      
      
    

    
    

    

    
hint
This is a counting problem, not a search. Each amino acid is coded by some
      number of codons; multiply those counts together. The number explodes fast, so take % 1000000
      as you go, not at the end. collections.Counter(codon_map.values()) gives you the degeneracy of
      each amino acid for free — and codon_map's 'Stop' entries count the stop codons.
