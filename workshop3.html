01

Why Python

You've got a shell and a repo. Now the language that turns them into science.

Python is…

High-levelyou don't manage memory — you describe the idea
Interpretedruns without compiling; you can poke at it line by line
Readableclear, minimal syntax — good Python often reads like pseudocode
Cross-platformthe same code runs on your laptop and on Fiji

Why scientists use it

Large ecosystemNumPy, pandas, SciPy, BioPython, scikit-learn — someone has already written the hard part
Fast prototypingtry an idea before lunch, iterate after
Communityvast user base, excellent docs, an answered Stack Overflow question for nearly everything
Glue languagebridges C++, Java, R, and bash — it holds pipelines together
Today you're handed a real knockout experiment: an RNA-binding protein called ZFP36L2 — which normally grabs certain mRNAs and marks them for destruction — deleted across six mouse tissues. Nobody's going to tell you what it does. In the lab — which opens in Google Colab — you'll load the actual sequencing results, hunt for patterns, and find out for yourself, then push your analysis to GitHub. Real bioinformatics, real data, one hour.

Speak Python

Third language, same lesson. Bash gave you imperative verbs; git gave you two-word verbs. Python gives you full sentences — with a subject, a verb, and a rule that whitespace is grammar. Tap any word to hear it in English.

The little words — Python's particles

= vs ==“let this name mean…” vs “are these equal?” — the most common beginner bug in any language
: + indentthe colon opens a block; the indentation is the block. Python has no braces — whitespace is syntax
& · |set math: a & b = in both (intersection) · a | b = in either (union). Today's whole hunt is one long &
#“ignore the rest of this line” — a note to humans
f'…{x}…'an f-string: drop the value of x right into the text — f'{gene}: up in {n} tissues'
.method()“ask this object to do something to itself” — line.strip(), row.split(','), genes.add(name)
The one to burn in today is &. Six tissues, five ampersands: lung & liver & bm & spleen & ovary & kidney — and whatever survives is what ZFP36L2 never lets go of, anywhere.

Logic & conditions

    print('never prints')
elif x or y:
    print('this one does')
else:
    print('only if none of the rest do')
Loops

    print(prime)

for i in range(10):     # 0..9
    if i == 5:

    print(i)

x = 0
while x < 4:
    print(x)
    x += 1              # same as x = x + 1

Slicing
# str[start:end:step]
hello[:5]     # 'Hello'      first five
hello[3:9]    # 'lo, Wo'
hello[7:]     # 'World'      from 7 to the end
hello[::-1]   # 'dlroW ,olleH'   reversed!

f-strings

Lists
