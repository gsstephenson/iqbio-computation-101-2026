Why this order

The Arc — skills that stack

The sequence is the argument: you can't use GitHub without a shell, you can't write real code without version control, and AI workflows are only as good as the foundations underneath them. Each session's output is the next session's prerequisite.

The sessions

Four workshops, four friction points killed

Every session is built around the specific place new grad students historically get stuck — and removes it while instructors are in the room. Tap a card's button to open its hands-on page.

01George & Nick

Bash & SSH / Cluster Computing

Thu Aug 13 · 2:00 – 4:00 PM · JSCBB A400

Friction it kills

"I've never touched a terminal, and the cluster is a black box."

You'll walk out able to

  • Log into fiji.colorado.edu from any laptop
  • Navigate and manage files fluently in bash
  • Know where files live — home vs. scratch vs. projects — and why it matters
  • Explain login vs. compute nodes and basic cluster etiquette

GitHub 101

Fri Aug 14 · 10:00 – 11:00 AM · JSCBB A400

Friction it kills

"SSH keys and GitHub config — the #1 place new students stall for weeks."

You'll walk out able to

  • Generate a keypair on Fiji, wired to your GitHub account
  • Pass the test: ssh -T git@github.com greets you by name
  • Explain repos, commits, branches, and remotes
  • Clone, push, and pull without fear

Python & Programming Practice
