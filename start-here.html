Emailed to the cohort 1–2 weeks before Workshop 1 as their first assignment. Work the checklist, then prove it in Verify & Submit at the bottom. Check items off as you complete them — your progress is saved in this browser. Step-by-step help for every item lives in the Setup Guides.

0 / 6 complete

  • 1 · Fiji account requested & approved

    Request through BioFrontiers IT — longest lead time, so it goes first. Exact request route included in the email.

    proof: the Fiji proof line — auto-checked in Verify & Submit below
  • 2 · Duo two-factor working on your IdentiKey

    Cluster login can trigger Duo; without it you're locked out before you start.

    proof: successful Duo push
  • 3 · CU VPN installed — Cisco Secure Client

    Required. CU servers — Fiji included — are unreachable from off campus without it. Install Cisco Secure Client (formerly AnyConnect) via OIT's install instructions; server address: vpn.colorado.edu.

    proof: screenshot of connected client
  • 4 · Terminal check

    macOS/Linux: open Terminal. Windows: open PowerShell. Run ssh -V. Since we teach bash on Fiji, everyone gets an identical environment — Windows students need only a working SSH client, not WSL.

    proof: ssh -V output — auto-checked in Verify & Submit below
  • 5 · GitHub account created

    Professional username; personal email as primary (accounts outlive .edu addresses); add your .edu for free GitHub Education benefits.

    proof: username auto-checked in Verify & Submit below
  • 6 · Laptop with admin rights + charger

    Every workshop is hands-on. A laptop you can install software on — not an iPad.

    proof: just bring it
